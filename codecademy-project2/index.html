Semantic Elements in HTML

Element Tags

Tag Name Description
<article> Article Self-contain composition in a document.
<aside> Aside Represents a portion of a document whose content is indirectly related to the maincontent.
<details> Details Creates a disclosure widget with information visible when it's toogled open.
<figure> Figure Represents a self-contained content, potentially with a caption
<figcaption> Figcaption Represents a caption or legend about its parent element
<footer> Footer Represents the footer of its nearest section.
<header> Header Represents introductory content, typically nav aids.
<main> Main Represents the dominant content of the body.
<mark> Mark Represents text which is marked or highlighted for reference notation.
<nav> Nav Represents a section of a page whose purpose is to provide navigation links
<section> Section Represents a generic standalone section of a document
<summary> Summary Specifies a summary, caption, or legend for a details element
<time> Time Represents a specific period in time.

Some element attributes

Attribute Name Description
class Class Marks the element with a class name so it can be refferred to by CSS and JavaScript
id Id Marks the element with a id name so it can be refferred to by CSS and JavaScript.
title Title Contains a text representing advisory information related to the element it belongs to.
